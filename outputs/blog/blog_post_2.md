# How to Buy $TTS Token on Base in 5 Minutes

Temptation Token ($TTS) trades on Uniswap V2 on Base mainnet. This guide walks you through every step.

**TTS Contract:** `0x5570eA97d53A53170e973894A9Fa7feb5785d3b9`
**Uniswap V2 Pool:** `0x77Fe188379BEaAd3BCFb26c965c812CEa721ce68`
**Chain:** Base (chainId 8453)

---

## Step 1: Install MetaMask (or Use Coinbase Wallet)

Download [MetaMask](https://metamask.io) for Chrome/Firefox or the MetaMask mobile app. If you already have MetaMask, skip to Step 2.

Alternatively, **Coinbase Wallet** works perfectly — it has native Base support built in.

Create your wallet, back up your seed phrase (write it on paper — do not photograph or save digitally), and you're ready.

---

## Step 2: Add the Base Network to MetaMask

Base isn't added by default. Here's how to add it:

**Option A (Easy):** Go to [app.temptationtoken.io](https://app.temptationtoken.io) and click Connect Wallet. The app will automatically prompt you to add Base mainnet.

**Option B (Manual):**
1. Open MetaMask → click the network selector (top left)
2. Click "Add Network" → "Add a network manually"
3. Enter these details:
   - Network Name: `Base`
   - RPC URL: `https://mainnet.base.org`
   - Chain ID: `8453`
   - Currency Symbol: `ETH`
   - Block Explorer: `https://basescan.org`
4. Click Save

---

## Step 3: Get ETH on Base

You need ETH on Base to pay for transactions (gas fees) and to swap for TTS.

**Option A — Bridge from Ethereum:** Use the official Base bridge at [bridge.base.org](https://bridge.base.org). Connect your wallet, enter the ETH amount, and bridge. Takes ~20 minutes. Fees: ~$5-15 depending on Ethereum gas.

**Option B — Buy directly on Base via Coinbase:** If you have a Coinbase account, you can withdraw ETH directly to Base (select "Base" as the network when withdrawing). Zero bridge fees, instant.

**Option C — Buy ETH on any CEX → withdraw to Base:** Binance, Kraken, OKX, and others support Base withdrawals. When withdrawing ETH, select "Base" network.

Recommended starting amount: $20-50 of ETH. TTS transactions cost less than $0.01 each.

---

## Step 4: Swap ETH for $TTS on Uniswap

1. Go to [app.uniswap.org](https://app.uniswap.org) (official Uniswap interface)
2. Connect your MetaMask wallet
3. Make sure you're on the **Base** network (check the network selector top right)
4. In the "From" field: select ETH
5. In the "To" field: paste the TTS contract address: `0x5570eA97d53A53170e973894A9Fa7feb5785d3b9`
6. If prompted with "Unknown token" — click Import (this is normal for new tokens)
7. Enter the ETH amount you want to swap
8. Set slippage to **2%** (click the gear icon → Slippage tolerance → 2%)
   - TTS has a 1% transfer tax, so 1% slippage minimum is required
9. Click Swap → confirm in MetaMask

Your $TTS will appear in your MetaMask wallet within seconds.

**Alternatively:** Buy directly inside the app at [app.temptationtoken.io](https://app.temptationtoken.io) — the Buy/Sell tab links directly to the correct Uniswap pool.

1. Open MetaMask → scroll down → click "Import tokens"
2. Paste contract: `0x5570eA97d53A53170e973894A9Fa7feb5785d3b9`
3. Symbol: TTS | Decimals: 18
4. Click Import

Your balance appears instantly.

## Frequently Asked Questions

**What's the minimum buy?** There's no minimum. You could buy $1 worth of TTS.

**Why does my transaction say "tax"?** TTS has a 1% transfer tax on all buys and sells. This is hardcoded in the smart contract and funds the prize ecosystem. Set 2% slippage to account for it.

**Is there a price impact warning?** For small buys (<$100), expect <1% price impact. For large buys, check the pool at [DexScreener](https://dexscreener.com/base/0x77Fe188379BEaAd3BCFb26c965c812CEa721ce68) first.

**Can I buy on a centralized exchange?** TTS is not currently listed on CEXes. Uniswap on Base is the only official market.

**Is the liquidity locked?** Yes — locked for 12 months on Team.Finance. You can verify this on BaseScan.
